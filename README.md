# 🎯 专业抽奖系统 - 企业级营销解决方案

一个功能完整、高度可定制的抽奖系统，专为企业营销活动、品牌推广、用户互动而设计。支持多种抽奖方式、主题切换和用户登录管理，助力您的营销活动取得成功！

## ⚙️ 配置说明

### 奖品配置

编辑 `src/js/config.js` 中的 `prizes` 数组：

```javascript
prizes: [
  {
    name: "一等奖",
    amount: "500元",
    probability: 0.05, // 5% 概率
    color: "#ffd700",
    icon: "fas fa-crown",
  },
];
```

### 主题配置

编辑 `src/js/config.js` 中的 `themes` 对象：

```javascript
themes: {
  default: {
    name: "默认主题",
    primary: "#667eea",
    secondary: "#764ba2",
    accent: "#ffd700",
    background: "linear-gradient(135deg, #667eea 0%, #764ba2 100%)",
    text: "#ffffff",
    card: "rgba(255, 255, 255, 0.1)"
  }
}
```

### 公司信息配置

编辑 `src/js/config.js` 中的 `company` 对象：

```javascript
company: {
  name: "您的公司名称",
  slogan: "您的公司口号",
  description: "公司简介",
  intro: "详细的公司介绍",
  contact: {
    phone: "联系电话",
    email: "邮箱地址",
    address: "公司地址",
    website: "公司网站"
  }
}
```

### 系统设置

编辑 `src/js/config.js` 中的 `system` 对象：

```javascript
system: {
  defaultLotteryType: "grid",  // 默认抽奖类型
  defaultTheme: "default",     // 默认主题
  showAudioControl: true,      // 显示音效控制
  showHistory: true,           // 显示抽奖记录
  showCompanyInfo: true        // 显示公司信息
}
```

## 🔧 自定义开发

### 添加新的抽奖方式

1. 在 `src/js/lottery-engine.js` 中添加新的抽奖方法
2. 在 `src/js/config.js` 中添加对应的配置
3. 在 `generateLottery()` 方法中添加新的 case

### 添加新的主题

1. 在 `src/js/config.js` 的 `themes` 对象中添加新主题
2. 确保主题包含所有必要的颜色变量

### 修改界面样式

1. 编辑 `src/css/styles.css` 文件
2. 使用 CSS 变量来支持主题切换

## 📦 部署说明

1. 将所有文件上传到 Web 服务器
2. 确保所有资源路径正确
3. 测试所有功能是否正常工作
